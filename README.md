## 🎯 Problema

Retener a un cliente existente cuesta hasta 5 veces menos que adquirir uno nuevo. Este proyecto analiza qué variables del historial de compras predicen si un cliente volverá a comprar, permitiendo priorizar acciones de fidelización.

---

## ✨ Qué incluye

- 🔍 **EDA completo** — distribuciones, correlaciones, valores nulos, outliers
- 📊 **Visualizaciones** de patrones de recompra por segmento
- 🤖 **Modelo predictivo** para clasificar propensión de recompra
- 📋 **Insights accionables** para estrategias de retención
