[GitHub](https://github.com/MedinaMarco/Analisis_Recompra)

</div>

---

## 🎯 Problema

Retener a un cliente existente cuesta hasta 5 veces menos que adquirir uno nuevo. Este proyecto analiza qué variables del historial de compras predicen si un cliente volverá a comprar, permitiendo priorizar acciones de fidelización.

---

## ✨ Qué incluye

- 🔍 **EDA completo** — distribuciones, correlaciones, valores nulos, outliers
- 📊 **Visualizaciones** de patrones de recompra por segmento
- 🤖 **Modelo predictivo** para clasificar propensión de recompra
- 📋 **Insights accionables** para estrategias de retención

---

## 🚀 Uso rápido

```bash
git clone https://github.com/MedinaMarco/Analisis_Recompra.git
cd Analisis_Recompra/Analisis_Recompra
pip install pandas matplotlib seaborn scikit-learn jupyter
jupyter notebook
```

# README — Dashboard Introvertido/Extrovertido

# 🧩 Dashboard de Análisis de Personalidad

**Introvertido vs. Extrovertido — Patrones de comportamiento desde los datos**

[![R](https://img.shields.io/badge/R-4.0+-276DC3?style=for-the-badge&logo=r&logoColor=white)](https://r-project.org)
[![ggplot2](https://img.shields.io/badge/ggplot2-Visualización-FC8D62?style=for-the-badge)](https://ggplot2.tidyverse.org)

> Dashboard analítico en R que explora diferencias de comportamiento entre perfiles de personalidad a través de datos de encuestas.

[GitHub](https://github.com/MedinaMarco/Introvertido_extrovertido_dashboard-Marco-Medina)

</div>

---

## 🎯 Contexto

Los perfiles de personalidad influyen en patrones de comportamiento medibles: hábitos sociales, comunicación, toma de decisiones. Este proyecto visualiza esas diferencias a partir de datos reales de encuestas.

---

## ✨ Qué incluye

- 📊 **Visualizaciones comparativas** entre grupos introvertido/extrovertido
- 🔍 **Análisis de variables** de comportamiento social, comunicación y preferencias
- 📋 Dashboard interactivo con filtros por variable

---

## 🚀 Uso rápido

```r
# Instalar dependencias
install.packages(c("ggplot2", "dplyr", "shiny", "flexdashboard"))

# Ejecutar dashboard
rmarkdown::run("proyecto_dashboard-Marco-Medina/dashboard.Rmd")
```

---

## 💡 Por qué R en lugar de Python

Este proyecto fue una oportunidad de aplicar el ecosistema de visualización de R (`ggplot2`, `tidyverse`) que, para análisis estadístico exploratorio, ofrece una sintaxis declarativa muy expresiva. Demuestra versatilidad en el stack de herramientas.
